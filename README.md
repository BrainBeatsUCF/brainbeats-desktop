## BrainBeats

### Development
You will need `npm` to install, run and build this project.

#### Ports
During development, the various parts of the application run in the following ports:
1. React: `localhost:3232`
3. Python Server: `localhost:5000`

#### Install
```npm install```

#### Run
```npm start```
